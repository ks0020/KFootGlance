# KFootGlance

## 👋 해외 축구 리그 소식과 한국인 선수들의 활약상을 담고 정보 공유를 위한 웹사이트

개발자 교육을 듣고 첫번째 프로젝트를 마친 후 복습과 복기를 하기 위한 새롭게 시작한 웹사이트 입니다.
